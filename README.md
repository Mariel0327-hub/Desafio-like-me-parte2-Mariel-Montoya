# Desafío Like Me - Parte II

Aplicación Full Stack desarrollada con:

- Frontend: React + Vite
- Backend: Express
- Base de datos: PostgreSQL

## Funcionalidades implementadas

- Obtener posts (GET)
- Crear posts (POST)
- Incrementar likes (PUT)
- Eliminar posts (DELETE)
- Manejo de errores con try/catch

## Instrucciones de ejecución

### Backend

cd server  
npm install  
npm run dev  

### Frontend

cd client  
npm install  
npm run dev  

Aplicación disponible en: http://localhost:5173
